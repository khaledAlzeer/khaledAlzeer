I'm a 3rd-year **AI & Data Science** student at Al-Zaytoonah University of Jordan (ZUJ), focused on turning raw data into intelligent, working systems. I move across the full stack — from cleaning a messy dataset, to training a model, to shipping it behind a real web app.
